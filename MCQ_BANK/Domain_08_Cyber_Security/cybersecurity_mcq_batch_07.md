### Question 601

Domain: Cyber Security
Topic: Wireless & Mobile Security
Subtopic: Wireless Encryption Protocols

Difficulty: Medium

Question: Which wireless security protocol uses CCMP for encryption and provides the strongest security for modern Wi-Fi networks?

A) WEP
B) WPA
C) WPA2
D) WPS

✔ Correct Answer: C

Reason: WPA2 uses CCMP (Counter Mode with Cipher Block Chaining Message Authentication Code Protocol) based on AES encryption, providing the strongest security among the options. WEP is outdated and vulnerable, WPA uses TKIP which is less secure, and WPS is a setup protocol, not an encryption protocol.

### Question 609

Subtopic: WEP Vulnerabilities

Difficulty: Hard

Question: Which cryptographic weakness in WEP makes it vulnerable to attacks?

A) Short key length and IV reuse
B) Weak password policy
C) No encryption
D) Open authentication

✔ Correct Answer: A

Reason: WEP's main cryptographic weakness is its short 24-bit initialization vector (IV) which leads to IV reuse after a relatively small number of packets. This, combined with RC4 stream cipher vulnerabilities, makes WEP easily crackable.
